Display the Contents of a Text/HTML File or any other text file as a widget
== Description ==

Display the Contents of a Text/HTML File or any other text file as a widget. This is useful when you are generating some data in the background as part of a cron process etc and now you want to display it through wordpress.

You can read more about it here => http://www.humbug.in/2011/wordpress-plugin-display-file-contents-widget-for-your-sidebar/

== Installation ==

1. Upload the `display-file-contents-widget` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer
3. Go to the Widgets Screen and Add the "Dislplay File Contents Widget"
4. Add a Title (Optional) and the path to the file to be displayed. It can be any text/html file.
5. The File Path will not be saved, till a valid path is provided (File should exist)
